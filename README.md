# ds230-manufacturing-trends
Class project
Overview

This project analyzes trends in the U.S. Manufacturing sector using data from the
Bureau of Labor Statistics (BLS) API.

The project follows a full Data Engineering + Data Analysis workflow:

API data extraction

JSON cleaning using Python & regex

Processed CSV creation

SQLite database creation

SQL queries

Jupyter Notebook analysis

Data visualizations and interpretation

This project uses GitHub Codespaces, Python, SQLite, and Jupyter Notebook.
